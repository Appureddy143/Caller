# AdmissionConnect — Telecaller College Admission System

## Setup: Neon PostgreSQL
1. Create account at neon.tech
2. Run schema.sql in the SQL Editor
3. Default admin: admin@college.com / Admin@123
4. Edit includes/config.php with your Neon credentials

## Deploy to Render
- Push to GitHub, connect to Render Web Service
- Set env vars: DB_HOST, DB_NAME, DB_USER, DB_PASS

## Roles: admin | telecaller | office
## First login: use system password → prompted to set new password
## Forgot password: verify with Date of Birth
